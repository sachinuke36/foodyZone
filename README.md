## Getting Started

## .env
```bash
NEXT_PUBLIC_STRIPE_PUBLIC_KEY 
STRIPE_SECRET_KEY 
MONGO_URI
```
## .env.local
```bash
AUTH_SECRET
AUTH_GOOGLE_ID
AUTH_GOOGLE_SECRET
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
